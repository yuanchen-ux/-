# sygydx-app
##
这是一个简易的购物网站，实现的功能有购物车模块，登录模块，商品预览模块等


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
